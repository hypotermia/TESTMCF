HOW TO RUN  
1. EXECUTE DATABASE YANG ADA PADA PROJECT DENGAN NAMA DBSqlQueryTOExecute.sql yang ada didalam file ini .
2. PLEASE SET THE PROJECT BECOME MULTIPLE PROJECT 
	- KLIK KANAN PADA SOLUTION NYA LALU PROPERTIES
	- LALU PILIH MULTIPLE STARTUP PROJECTS , DAN UNTUK DI BAGIAN ACTION PILIH START, 
3. SETTING CONNECTION STRING YANG ADA PADA appsettings.json di BackendAPI sesuaikan nama database dan server .
	CONTOH : 
	 "ConnectionStrings": {
   "DefaultConnection": "Server=localhost;Database=MasterDB;Trusted_Connection=True;MultipleActiveResultSets=true"
 		}
	Note * JANGAN LUPA GANTI SERVER DISESUAIKAN DENGAN SERVER ANDA


UNTUK TEST YANG QUERY TEST ADA DI FOLDER INI DENGAN NAMA ANSWER QUERY TEST .
